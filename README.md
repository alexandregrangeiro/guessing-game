# Guessing Game

A simple number guessing game written in C, intended to be played in the terminal.

## How to compile and run

Compile on terminal, then run on terminal. May as well use your IDE. Using gcc:

```bash
gcc src/guessing.c -o guessing //compiling
./guessing //running
```
